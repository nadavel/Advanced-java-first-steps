Project requirements were:
1. create a trivia game
2. read questions and the 4 answers from a file where the first answer is the correct one
3. on each turn the player has 10 seconds to answer
4. correct answer rewards 10 points and an incorrect one (or time out) decreases 5 points
5. questions will be shown in random order
6. the gui will indicate correct/incorrect answer
7. allow the player to end the game and to start a new game via buttons
8. game ends when player chooses to finish the game or when the questions run out
9. when the game ends show the player his score/success rate
10. the gui will allow only a single answer to be picked by the player for each question (no cheating!)
